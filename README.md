## Divergensity

The "Reheboam" AI interface from the Westworld TV series.

![Preview](./public/Picture.JPG)

- Watch critical COVID-19 timeline events.
- Create and launch a custom timeline.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Type Check

```bash
npm run typecheck
```
